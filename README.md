# NS3-AODV-Optimized-Route-Discovery-Using-Dbscan
Optimized route discovery for AODV routing algorithm using Dbscan clustering


### Steps for running the examples
1. Download and install [ns3.46](https://www.nsnam.org/releases/ns-3-46/)
2. Copy the folder ![aodvDbscan](/aodvDbscan) to **ns3/src** or **ns3/contrib**
3. Build the new module **aodvDbscan** [Instructions](https://www.nsnam.org/docs/manual/html/new-modules.html)
4. Add the examples from ![scratch](/scratch) to **ns3/scratch**
